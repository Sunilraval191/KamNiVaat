# 🚀 KamNiVaat - Local Job Portal for Kalol

**KamNiVaat** (Work Talk) is a professional, community-focused job portal connecting local businesses with job seekers in Kalol and Gandhinagar.

## 📈 SEO & Growth Strategy

### 1. SEO Metadata
- **Portal Name:** KamNiVaat
- **Primary Keywords:** Jobs in Kalol, Kalol job portal, Local hiring Gandhinagar, Accountant jobs Kalol, Kam Ni Vaat, Kam ni vat.
- **Search Console:** Submit your GitHub URL to Google Search Console to rank for local job searches.

### 2. Viral Growth
- **WhatsApp Marketing:** Use the "Share on WhatsApp" button to push listings into city groups.
- **Offline Trust:** Print the portal's QR code on local flyers. Use the "Verified Business" badge in the database to highlight trusted employers.

### 3. Key Features
- **Newspaper Style:** Contact details visible as text for high trust.
- **Dynamic Posting:** Employers can post up to 5 roles in one submission.
- **Auto-Expiry:** Jobs automatically hide after 7 days to maintain portal quality.
